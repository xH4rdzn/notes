___
- O operador de atribuição é o `=`;
- E podemos usar ele para fazer operações aritméticas, como no exemplo a seguir:
```js
let value

// Operador de atribuição
value = 1
```
- Para fazer operações aritméticas:
```js
// Soma mais 2 a value
value += 2

// Subtrai 3 de value
value -= 3

// Multiplica
value *= 5

// Divisão
value /= 2

// Resto da divisão
value %= 2

// Potência
value **= 2
```
