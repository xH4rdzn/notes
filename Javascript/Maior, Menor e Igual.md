___
- Esses operadores serve para comparar se é maior, menor ou igual a;
- Operador maior que: `>`
```js
let balance = 500
let payment = 120

console.log(balance > payment)// true
```
- Operador menor que: `<`
```js
let balance = 500
let payment = 120

console.log(balance < payment) // false
```
- Podemos também verificar se é maior ou igual a: `>=`;
```js
let balance = 120
let payment = 120

console.log(balance >= payment) // true
```
- Menor ou igual a : `<=`
```js
let balance = 500
let payment = 120

console.log(balance <= payment) // false
```
