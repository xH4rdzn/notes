___
- Podemos atualizar propriedades de um objeto, para fazer isso podemos seguir o exemplo abaixo:
```js
const product = {
	name: "Teclado",
	quantity: 100,
}

product.quantity = 90
```
- Desta maneira conseguimos atualizar qualquer propriedade do nosso objeto;
- Também podemos usar a notação de colchetes;
