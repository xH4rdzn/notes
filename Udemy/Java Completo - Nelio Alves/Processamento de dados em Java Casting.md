___
- Processamento de dados é feito através do comando de atribuição
```text
Sintaxe:
<variavel> = <expressão>;
```
- Exemplos:
```java
// Exemplo 1
int x, y;

x = 5;

y = 2 * x;

System.out.println(x);
System.out.println(y);
```
```java
// Exemplo 2
int x;
double y;

x = 5;

y = 2 * x;

System.out.println(x);
System.out.println(y);
```
```java
// Exemplo 3
double b, B, h, area;

b = 6.0;
B = 8.0;
h = 5.0;

area = (b + B) / 2.0 * h;
System.out.println(area);
```
```java
// Exemplo 4

int a, b;
double resultado;

a = 5;
b = 2;

// Casting
resultado = (double) a / b;

System.out.println(resultado);
```
