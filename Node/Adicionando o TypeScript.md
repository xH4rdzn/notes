___
- Para adicionar o typescript no nosso projeto vamos ter que instalar ele, como só vamos usar em ambiente de desenvolvimento, vamos instalar ele como uma dependência de desenvolvimento;
- Então para instalar o `typescript`, usamos o comando:
```bash
npm i typescript -D
```
- E também precisamos instalar a tipagem do Node, e instalamos usando o comando:
```bash
npm i @types/node -D
```
- Após fazer a instalação dos comandos acima, irá ser gerado uma pasta com o nome de: `node_modules`;
- Essa pasta é onde ficará todas as dependências do nosso projeto;
