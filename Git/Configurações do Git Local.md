___
- Toda vez que fazemos a instalação do *Git*, precisamos fazer a sua configuração local.
- Para configurar o nome, usamos o comando abaixo:
```zsh
git config --global user.name "Seu nome"
```
- Para configurar o seu e-mail, usamos o comando:
```zsh
git config --global user.email "email@email.com"
```
- Podemos ver todas as configurações com o comando:
```zsh
git config --list
```
