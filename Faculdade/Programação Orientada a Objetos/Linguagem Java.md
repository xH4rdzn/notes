### História do Java
- Surgiu nos anos 1990
- Na empresa *Sun Microsystems*
- No projeto *Green Project*
	- Responsável por projetar o *StarSeven*, que é um aparelho que era focado em controle domestico.
	- Precisava funcionar em diversos equipamentos, então surgiu o conceito de Máquina Virtual (Programa que roda programas).
	- Linguagem Oak
		- Esse nome acabou por questões legais.
	- Conectividade doméstica
- Por **James Gosling**

### Organização do Java
- Java é uma linguagem que podemos dizer que é *compilada* e *interpretada*.
- Ela é *compilada* para ***bytecode***, que por sua vez é interpretada pela *Máquina Virtual*
