___
- Para poder aplicar herança entre classes, usamos a ***keyword*** ***`extends`***, dessa maneira todos atributos e métodos da classe herdada são passados;
```java
public class ContaInvestimento extends Conta {
	
	// Métodos específicos da Conta Investimento
	
}
```
- Importante que de maneira implicita, todas as classes herdam da classe ***`Object`***;
