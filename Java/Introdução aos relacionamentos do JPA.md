### Porquê precisamos deles ?
- Os relacionamentos existem para podermos estruturar nossos dados com uma maior proximidade com a realidade

### Quais são os tipos ?
- Os relacionamentos entre tabelas de banco de dados são representados através de `@annotations`.
- Que são:
- ***`@OneToOne`*** -> Ex: Uma pessoa tem exatamente um Passaporte.
- ***`@OneToMany`*** -> Ex: Uma categoria pode ter muitos Produtos.
- ***`@ManyToOne`*** -> Ex: Muitos Produtos pertencem a uma Categoria.
- ***`@ManyToMany`*** -> Ex: Muitos estudantes podem se inscrever em muitos Cursos.
