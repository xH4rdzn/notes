- Usamos as classes abstratas, para evitar que essa classe não possa ser instanciada, apenas herdada;
- Para tornar uma classe em abstrata, usamos a keyword ***`abstract`***, como no exemplo abaixo:
```java
public abstract class NotaFiscal {
	
	// Implementação
	
}
```
- Nessa classe, podemos ter métodos e variáveis de instância normalmente;
