- Quando trabalhamos com [[Implementando herança|herança]], podemos nos deparar com métodos que possuem diferentes comportamentos de acordo com cada classe;
- Para implementar uma lógica específica, podemos ***sobrescrever os métodos***;
- Para podermos fazer isso, devemos copiar o método de maneira que o **nome e seus parâmetros** fiquem iguais ao da classe que está sendo herdada;
